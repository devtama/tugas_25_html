Mengenal Semantic Elements di HTML5

  • Articel

    • Ketika ingin membuat layout untuk menampilkan block artikel-artikel yang ada di website kita, tag article bisa kita gunakan dengan element header dan footer.
    Karena type article memiliki type block, artinya type ini memiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya.

  • Aside

    • Element ini digunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside dibagian bawah postingan untuk menjelaskan identitas penulis artikel. Karena aside memiliki sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
